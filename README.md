# OREH
독립유공자의 더 나은 삶을 꿈꾸는 단체, OREH에서 만든 역사 관련 퀴즈 애플리케이션입니다.\
개발 외주를 받아, OREH 측에서 제공해준 figma 디자인대로 애플리케이션을 제작하였습니다.\
React-native로 앱을 개발하였으며, android와 ios 2개의 플랫폼에 대하여 애플리케이션을 제작하여 play store와 apple store에 배포하였습니다.
